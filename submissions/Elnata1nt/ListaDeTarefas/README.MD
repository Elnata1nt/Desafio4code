# Lista de Tarefas

Este é um projeto simples de Lista de Tarefas que permite adicionar, editar, marcar como concluídas e limpar tarefas concluídas. Além disso, você receberá notificações para lembrar-se de concluir as tarefas a tempo.

## Funcionalidades

- Adicionar uma nova tarefa com uma data e hora de conclusão.
- Editar o texto da tarefa.
- Marcar tarefas como concluídas.
- Limpar tarefas concluídas.
- Receber notificações no horário de conclusão das tarefas.

## Estrutura do Projeto

- `index.html`: Estrutura HTML do projeto.
- `listas.css`: Estilos CSS para o projeto.
- `listas.js`: Lógica JavaScript do projeto.

## Como Usar

1. Clone o repositório ou faça o download dos arquivos.
2. Abra o arquivo `index.html` em um navegador.
3. No campo "Adicione uma tarefa", insira o texto da tarefa.
4. No campo de data e hora, insira o prazo para conclusão da tarefa.
5. Clique em "Adicionar Tarefa" para adicionar a tarefa à lista.
6. Clique na tarefa para marcá-la como concluída.
7. Clique no botão "Editar" ao lado da tarefa para editar o texto da tarefa.
8. Clique em "Limpar Tarefas Concluídas" para remover todas as tarefas marcadas como concluídas.
